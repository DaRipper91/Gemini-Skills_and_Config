# Config

This directory stores local Gemini CLI configuration and memories.

- `GEMINI.md` – Saved context and notes the agent should remember.
- `google_accounts.json` – Google account information for authentication.
- `settings.json` – CLI settings.
- `state.json` – Cached state for active sessions.
- `trustedFolders.json` – Folders trusted for tool execution.
- `GEMINI.md.bak` – Backup of the primary memory file.

Update these files to match your environment before running the CLI.
