# Extensions

This directory houses Gemini CLI extensions that add tools for research, documentation, coding assistance, and automation. Each subfolder contains the extension source plus its own README or docs.

Common categories:
- **Research & docs**: `gemini-deep-research`, `co-researcher`, `scribe`, `gemini-cli-prompt-library`.
- **Developer tooling**: `code-review`, `git-expert`, `github`, `plan`, `run-long-command`.
- **System & automation**: `ComputerUse`, `adb-control-gemini`, `conductor`, `mcp-toolbox`.
- **Branding & assets**: `gemini-logos`.
- **Repo packaging**: `repomix-manager`.

See the README inside a specific extension for setup and usage details.
